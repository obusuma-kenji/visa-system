# 🚀 外国人雇用在留資格診断システム - クイックスタートガイド

## 📦 すぐに始める（3分で起動）

### ステップ1: フォルダを開く
```bash
cd foreign_worker_visa_system
```

### ステップ2: サーバーを起動
```bash
python manage.py runserver
```

### ステップ3: ブラウザでアクセス
```
http://127.0.0.1:8000/
```

**これだけで動作します！**
（データベースと初期データは既に設定済み）

## 📝 使い方（30秒チュートリアル）

### 一般ユーザー向け

1. トップページの「診断を開始する」をクリック
2. フォームに情報を入力：
   - 国籍: ベトナム
   - 学歴: 学士（大学卒業）
   - 専攻: 情報工学
   - 業種: IT・ソフトウェア
   - 職種: システムエンジニア
   - 月額報酬: 280000
3. 「診断する」ボタンをクリック
4. 結果を確認！

### 管理者向け

1. 管理画面にログイン
2. 「在留資格」「在留資格要件」などを編集可能
3. 「診断セッション」で過去の診断履歴を確認

## 🎯 実装済み機能一覧

✅ **在留資格診断エンジン**
- 自動スコアリング（0-100点）
- 要件チェック（学歴、経験、報酬など）
- 複数候補の比較表示

✅ **6種類の在留資格に対応**
1. 技術・人文知識・国際業務（最も一般的）
2. 特定技能1号（14分野）
3. 特定技能2号
4. 高度専門職
5. 技能
6. 企業内転勤

✅ **18種類の業種・職種マッピング**
- IT、製造、商社、飲食、建設、介護など

✅ **必要書類リスト自動生成**
- 在留資格ごとに必要書類を表示

✅ **診断履歴の保存**
- 過去の診断を管理画面で確認可能

## 🔐 セキュリティ設定（本番環境用）

```python

# 1. SECRET_KEYの変更
SECRET_KEY = 'your-new-secret-key-here'

# 2. DEBUGをOFFに
DEBUG = False

# 3. ALLOWED_HOSTSの設定
ALLOWED_HOSTS = ['yourdomain.com', 'www.yourdomain.com']

# 4. データベースをPostgreSQLに変更
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'visa_db',
        'USER': 'your_user',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
