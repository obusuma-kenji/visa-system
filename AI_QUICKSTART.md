# 🤖 AI機能 - 5分でセットアップ

## 📥 Step 1: 必要なパッケージをインストール

```cmd
pip install anthropic
```

---

## 🔑 Step 2: APIキーを取得

1. https://console.anthropic.com/ にアクセス
2. Sign Up（初めての場合）
3. 左メニュー「API Keys」→「Create Key」
4. APIキーをコピー（`sk-ant-...`で始まる）
5. 「Billing」から$5-10のクレジット追加

---

## ⚙️ Step 3: settings.pyを編集

`visa_system/settings.py`の最後の方を編集：

```python
# AI統合設定
ANTHROPIC_API_KEY = 'sk-ant-ここに取得したAPIキーを貼り付け'  # ← 編集

# AI機能の有効化フラグ
ENABLE_AI_FEATURES = True  # ← FalseをTrueに変更
```

---

## ✅ Step 4: 動作確認

サーバーを起動:
```cmd
python manage.py runserver
```

起動ログに以下が表示されればOK:
```
✅ AI機能が有効化されました
```

---

## 🧪 Step 5: テスト診断

1. http://127.0.0.1:8000/diagnosis-form/ にアクセス
2. 以下のテストデータを入力:

```
国籍: ベトナム
最終学歴: 学士（大学卒業）
専攻: 経営学
実務経験: 3年
経験分野: 営業
業種: 商社・貿易
職種: 海外営業
業務内容: 海外顧客との英語での商談、契約書作成、市場調査と販売戦略の立案
月額報酬: 280000
```

3. 「診断する」をクリック
4. 結果ページに **🤖 AI詳細分析** セクションが表示される

## 🔧 トラブルシューティング

### エラー: `AuthenticationError`
→ APIキーを確認（`sk-ant-`で始まっているか）

### AI分析が表示されない
→ サーバー起動時のログに「✅ AI機能が有効化」と出ているか確認

### `ModuleNotFoundError: anthropic`
→ `pip install anthropic` を実行
